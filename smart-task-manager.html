    The project

    A web app where you write your daily tasks like a normal list — except it's not just a list. An AI organizer reads each task and sorts your day by real priority instead of the order you typed them in.

    The benefit

    Brings together a working UI, persistent state, and an AI integration layer that reasons about unstructured text and returns structured data the interface can render.
